# EarAid

A Celeste mod that lets you define custom groups of sounds and adjust their volume individually. sorry kevin

## How it works

EarAid 2.0 uses **Sounds** — named collections of one or more FMOD events that share a single volume slider 
(0%–200%, default 100%).

### Your Sounds

Shows each of your saved Sounds with a volume slider.

### Manage Sounds

Opens an overlay where you can browse saved Sounds, add a new Sound, rename a Sound, edit its events, or delete it.

Note that the event search uses the game's naming scheme for events, which may differ from what the "accepted" name is. Ask
around if you're struggling to find the sound you're looking for.

## Upgrading from 1.x

On first launch after updating, any 1.x volume settings that were not at 100% are automatically converted into sounds
with the same names and volumes.
